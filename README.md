This project is a complete and production-ready integration with SSL Wireless Push SMS Gateway API v3.0.0 — the leading SMS service provider in Bangladesh.
It enables sending SMS in three modes from a single codebase:

Single SMS – Send one message to one recipient
Bulk SMS – Send the same message to up to 100 recipients in one request
Dynamic SMS – Send personalized (different) messages to up to 100 recipients at once

Key Features:

Full support for English & Bangla (Unicode) messages (auto-detected)
Proper error handling with detailed API & SMS status codes
Unique csms_id generation (daily unique as required by SSL)
MSISDN validation & formatting (88019xxxxxxxx format)
Rate limit & duplicate protection
Clean, reusable, and well-documented code
Ready for Laravel, plain PHP, or any framework

Used for: OTP, transactional alerts, promotional campaigns, notifications, etc., across all operators in Bangladesh (Grameenphone, Robi, Banglalink, Teletalk).
Official Gateway: https://sms.sslwireless.com
Support: service.operation@sslwireless.com
Perfect for e-commerce, banks, fintech, ride-sharing, food delivery, and any business needing reliable SMS delivery in Bangladesh.
